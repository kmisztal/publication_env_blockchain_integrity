# Proof-of-Concept Experiments

This package contains executable proof-of-concept experiments for the paper.
The first MVP dataset path is OpenAQ, with reusable experiment utilities kept in
`experiments/common/` and OpenAQ-specific ingestion kept in `experiments/openaq/`.

## Current Scope

- Ingest a frozen local OpenAQ CSV, JSON, or JSONL export.
- Normalize records into a canonical measurement schema.
- Create a dataset manifest with file hashes and source metadata.
- Optionally load the manifest and normalized measurements into SQLite.

No experimental threat-coverage results are generated by ingestion.

## Environment

Use the project virtual environment:

```powershell
e:\git_venv\dicella\env_blockchain_integrity\Scripts\Activate.ps1
pdm install
```

Alternatively, point PDM at the existing interpreter:

```powershell
pdm use e:\git_venv\dicella\env_blockchain_integrity\Scripts\python.exe
pdm install
```

Package management is handled through PDM. The current `pyproject.toml` declares the minimal ingestion dependency set. Avoid creating a project-local `.venv`; the intended environment is the shared venv above.

## Example

```powershell
pdm run openaq-ingest `
  --source-file path\to\openaq_export.csv `
  --dataset-id openaq_mvp `
  --source-url https://docs.openaq.org/ `
  --query-parameters-json "{}"
```

Outputs are written under `experiments/data/` by default.
